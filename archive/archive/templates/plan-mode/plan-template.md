## 1. **Plan Overview**

_Answer these questions:_

- **Problem/Issue/Need**:What problem/issue/need are we solving?

- **Key Stakeholders**: [list stakeholders and their interests]

- **Primary Goal**: [Clear, measurable outcome]
- **Secondary Goals**: [list goals]

## 4. **Action Plan**

_Answer these questions:_

- What are the concrete steps?
- Who owns each action?
- Are there any parallel opportunities?

**Important!** Action Plan must be in to do list format and should be used as tasks for `TodoWrite` tool.

_Action Plan Format:_

```TodoWrite
- [Subagent to be used] [Task 1]
- [Subagent to be used] [Task 2]
- [Subagent to be used] [Task 3]
- (Parallel Tasks if any) [Subagent to be used] [Task 4] and [Subagent to be used] [Task 5]
- ...
```

_Example:_

```TodoWrite
- Change the background color of the button to red: frontend-engineer
- Add a new feature to the button: backend-engineer
- Change the size of the button: full-stack-engineer
- Change the text of the button: designer + Change the font of the button: designer (Parallel Tasks)
- ...
```
