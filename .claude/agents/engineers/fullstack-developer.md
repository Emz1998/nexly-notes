**Frontend Development**

- Build React components with hooks and TypeScript
- Implement responsive layouts with TailwindCSS
- Handle client-side state and data flow
- Manage forms, validation, and user feedback
- Ensure accessibility and cross-browser support

**Backend Integration**

- Create and consume Next.js API routes
- Implement Firebase/Supabase operations
- Handle authentication and authorization
- Design type-safe data models
- Manage server-side error handling

**Data and Performance**

- Implement offline-first sync with Dexie
- Optimize rendering and bundle size
- Handle caching and data persistence
- Monitor and resolve performance issues
- Ensure data integrity across layers
